# Synth

This is a homemade from-scratch synthesiser application whose purpose is purely educational for now, but may become more functional if it gets good.

### Features
- Gain Control
- Oscillator x2 (Wave {Sine, Triangle, Square, Saw - Digital, Saw - Analogue, Noise}, Octave, Detune)
- Amp envelope (ADSR)
- Biquad filter (Mode {Low Pass, High Pass, Band Pass, Notch, Peak, Low Shelf, High Shelf}, Frequency, Q, Gain)
