# SHA-A Archangel
### By StrongHand Audio

This is a homemade from-scratch modular synthesiser application whose purpose is purely educational for now, but may become more functional if it gets good.

## Components
### Working (mostly)
- Oscillator
- ADSR envelope
- Biquad Filter
- Mixer

## Planned
- Reverb
- Delay
- Compressor
- EQ
- Wavetable oscillator
- Flanger
- Phaser
- Chorus
